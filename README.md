# Rick-Tac-Toe
Ready for the next Rick and Morty season? I'am! So, I've decided to implement some of this series university style to this project. The main objective of it has been to learn and apply factory functions and modules (as well as closures, scopes, public/private variables). This knowledge will be helpful for having a better organized code. For the time being, this game is only computer-based and it's also a 1vs1 game; there is no AI implemented (my stubborness will make it happen).

# Choosing Characters
For choosing a character you will need to click on the character logo and then on either the player one or player two button. For the first container, player one. The second container, player two.
